Loading Croc: Legend of the Gobbos (PS1)

Preparing the game files. Track 1 is split into multiple parts due to file-size limits, so this page has to download and stitch it together before the emulator can boot.
If controls feel “dead” during loading, that is normal. The game will respond after the disc is mounted.
Tip: after it starts, click the game once so the keyboard focuses the emulator.
Keyboard: arrows = dpad, Z/X = buttons, Enter = start, Shift = select (varies by core).
